    PYSunday 其它API

    还有其它好用的api在这里展示

    cmdexec

    sunday.core.cmdexec.cmdexec(cmd, timeout=10, **kwargs)

    执行系统命令, 返回: code, stdout, stderr, 当code不为0则为报错 Args: cmd(str): 命令文本 timeout(int): 超时时间 Returns: code(int): 为0则执行成功，否则执行失败 success(str): 执行成功打印内容 error(str): 执行失败打印内容

    getConfig

    sunday.core.getConfig.getConfig(name, file='config.ini', default={})

    返回能够获取全局配置文件($SUNDAY_ROOT/config.ini)内配置值的函数 Args: name(str): 分类名 file(str): 文件名 default(dict): 默认配置的集合，优先级大于全局的配置 Returns: getval(key): 返回键对应的值 Usages: >>> getConfig('LOGGING')('level') DEBUG

    getvar

    sunday.core.globalvar.getvar(name)

    根据内部全局常量名返回值 Args: name(str): 全局常量键名

    setvar

    sunday.core.globalvar.setvar(name, value)

    设置内部全局常量值 Args: name(str): 全局常量键名 value(str): 全局常量值

    getMd5

    sunday.utils.md5.getMd5(value)

    文本生成md5值 Args: value(str): 文本字符串

    currentTimestamp

    sunday.utils.tools.currentTimestamp()

    返回当前时间戳, 单位为毫秒

    exit

    sunday.core.common.exit(text='')

    退出程序 Args: text(str): 程序退出并打印退出提示文本

    clear

    sunday.core.common.clear()

    清空屏幕

    pascal_to_snake

    sunday.core.common.pascal_to_snake(text)

    驼峰转蛇形 Usages: >>> from sunday.core.common import pascal_to_snake >>> pascal_to_snake('HelloWorld') hello_world Args: text(str): 驼峰串

    snake_to_pascal

    sunday.core.common.snake_to_pascal(text)

    蛇形转驼峰 Usages: >>> from sunday.core.common import snake_to_pascal >>> snake_to_pascal('hello_world') HelloWorld Args: text(str): 蛇形串

    parseJson

    sunday.core.common.parseJson(jsonPath, defaultValue={}, keywords=[], logger=None)

    传入json文件，返回解析后的json数据 Args: jsonPath(str): 文件路径 defaultValue(list | dict): 文件读取失败时返回的默认数据 keywords(list): 需要的键集合 logger(Logger): 日志对象

    parseJsonp

    sunday.utils.tools.parseJsonp(text, defvalue={})

    解析jsonp数据返回dict数据 Args: text(str): jsonp数据字符串, 格式如：jsonp_12345({})

    getEnv

    sunday.core.getEnv.getEnv(key)

    返回键名为key的系统环境变量值

    grenImagePng

    sunday.utils.image.grenImagePng(*args, **kwargs)

    生成指定宽高及颜色的png图片 Args: width(int): 图片宽 height(int): 图片高 color(str): 图片颜色，不传时为随机颜色

    grenImageJpeg

    sunday.utils.image.grenImageJpeg(*args, **kwargs)

    生成指定宽高及颜色的jpeg图片 Args: width(int): 图片宽 height(int): 图片高 color(str): 图片颜色，不传时为随机颜色

    sunday.utils.image.grenImageJpg(*args, **kwargs)

    生成指定宽高及颜色的jpg图片, 同grenImageJpeg Args: width(int): 图片宽 height(int): 图片高 color(str): 图片颜色，不传时为随机颜色

    sunday.utils.image.grenImageGif(*args, **kwargs)

    生成指定宽高及颜色的gif图片 Args: width(int): 图片宽 height(int): 图片高 color(str): 图片颜色，不传时为随机颜色
